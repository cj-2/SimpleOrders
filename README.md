# SimpleOrders

Esse projeto (hoje) basicamente implementa dois fluxos de produção e consumo de mensagens para brokers, Apache Kafka e RabbitMQ.

Configurei um arquivo para ser executado pelo docker compose (compose.yml) que faz a criação dos brokers, inicia a api e os workers (consumidores) em ponto de bala para ser testado.

Não é nada avançado, mas é uma estrutura para ser utilizada de base em meus próximos projetos. Espero que seja útil de alguma forma.
